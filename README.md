
# ImportlessApi

  ## ImportlessApi is a project that allow you to easly use real time function resolving using a changing hash value.

  ## Usage
  
  * include "ImportlessApi.hpp"
  * Handle hFile = IMPORTLESS_API(CreateFile)(Parameters);
